Embedding Code Snippets

Embedding a code snippet into a FubuMVC view is really pretty simple with the following page extensions:

  1.  
  2.         /// <summary>
  3.         /// Embed a code snippet into a FubuMVC view
  4.         /// </summary>
  5.         /// <param name="page"></param>
  6.         /// <param name="snippet"></param>
  7.         /// <returns></returns>
  8.         public static HtmlTag CodeSnippet(this IFubuPage page, Snippet snippet)
  9.         {
  10.             var assets = page.Get<IAssetRequirements>();
  11.  
  12.             assets.Require("prettify.js");
  13.             assets.Require("bootstrap-prettify.js");
  14.             assets.Require("prettify.css");
  15.  
  16.             return new SnippetTag(snippet);
  17.         }
  18.  
  19.         /// <summary>
  20.         /// Embed a named code snippet into a FubuMVC view
  21.         /// </summary>
  22.         /// <param name="page"></param>
  23.         /// <param name="snippetName"></param>
  24.         /// <returns></returns>
  25.         public static HtmlTag CodeSnippet(this IFubuPage page, string snippetName)
  26.         {
  27.             var snippet = page.Get<ISnippetCache>().Find(snippetName);
  28.             return page.CodeSnippet(snippet);
  29.         }

The usage for a Spark view is:

  1.         <h4>C#</h4>
  2.     !{this.CodeSnippet("AddLine")}
